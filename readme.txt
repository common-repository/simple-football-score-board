Generate football scoreboard from shortcode.

== Description ==

This plugin generate a simple football scoreboard. All you have to do is to put shortcode with some information of the game in your post, widget or fixed page.
You can customize color in the setting page.

== Installation ==

1. Upload the entire `ytmr_simple_football_scoreboard` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Is it possible to put background image? =

No, only background-color allowed.
